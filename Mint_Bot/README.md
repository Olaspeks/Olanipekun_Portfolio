
# Mint Bot + Mint Websites

**Description:**  
This project simulates a high-speed NFT minting bot with both smart contract and frontend components. It includes scripts to simulate mint transactions and dashboards for NFT projects.

**Tech Stack:** Solidity, OpenZeppelin, Python, JavaScript (React), Foundry

**Folder Structure:**
- `contracts/` - NFTMintBot.sol: ERC721 contract with minting automation
- `frontend/` - dashboard.js: Simulates user dashboard for minting
- `scripts/` - simulate_mint.py: Python simulation of minting transactions

**Role:** Builder / Smart Contract Developer

**Outcome:**  
- Developed professional NFT minting smart contract
- Created frontend simulation for dashboards
- Python scripts showcase automation logic
- Fully documented and recruiter-friendly
