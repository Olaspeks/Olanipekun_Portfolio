# Pendax - Crypto Payment Platform

**Description:**  
Pendax is a crypto payment platform for simple utility transactions. This project demonstrates backend payment processing using smart contracts and Python simulation scripts.

**Tech Stack:** Solidity, ERC20, Python

**Folder Structure:**
- `contracts/` - PendaxPayment.sol: ERC20-based payment smart contract
- `backend/` - crypto_payment_backend.py: Python simulation of backend transactions

**Role:** Backend Developer / Smart Contract Developer

**Outcome:**  
- Developed professional ERC20 payment smart contract with event tracking  
- Created Python scripts simulating payment processing  
- Demonstrates real-world crypto payment workflows and backend logic
